# Project Summary
The AI-Powered Healthcare Platform is a comprehensive web application designed to unify patient prescription histories across multiple doctors. It enhances patient safety by preventing medication interactions and providing a seamless experience for patients, doctors, and administrators. The platform features a public landing page that explains its functionalities, highlights key features such as verified doctors, unified medical history, AI medicine identification, reminders, and multilingual support. It integrates Google authentication for secure access and role-based management, ensuring that only verified healthcare professionals can access sensitive information.

# Project Module Description
- **Authentication Module:** Handles user login, role selection, and profile completion.
- **Doctor Module:** Allows doctors to create profiles, manage prescriptions, and view patient histories.
- **Patient Module:** Enables patients to create profiles, manage their medical history, and interact with an AI chatbot.
- **Admin Module:** Manages doctor verifications and oversees platform statistics.
- **Notification System:** Sends SMS alerts for various actions using Brevo API.
- **Chatbot Integration:** Provides AI-driven support for medication identification and health inquiries.
- **Landing Page:** A public page introducing the platform with sections on Home, Features, How It Works, Why Use This Platform, and a language selector (English/Tamil/Hindi) for accessibility.

# Usage
1. **Install Dependencies:** Run `pnpm install` to install project dependencies.
2. **Build the Project:** Use `pnpm run build` to compile the project.
3. **Run the Application:** Use `pnpm run start` to launch the application locally.
